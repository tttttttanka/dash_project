# Инструкция по запуску тестов

Для запуска тестов выполните следующие шаги:

1. **Запустите Dash-приложение**: Убедитесь, что ваше веб-приложение запущено локально (обычно по адресу `http://127.0.0.1:8050/`).
2. **Откройте проект в IntelliJ IDEA**: Откройте папку с тестами как Maven-проект.
3. **Запуск через TestNG XML**:
    - Нажмите правой кнопкой мыши на файл `testing_full.xml` в корне папки с тестами.
    - Выберите пункт **Run '.../testing_full.xml'**.
    - Нажмите правой кнопкой мыши на файл `testing_smoke.xml` вы корне папки с тестами.
    - Выберите пункт **Run '.../testing_smoke.xml'**.
4. **Чтобы запустить каждый тест по отдельности:**
    - Откройте файл DashTest.java.
    - Нажмите на запуск слева от каждого теста.
5. **Запуск через Maven (опционально)**:
   В терминале, находясь в папке с файлом `pom.xml`, выполните команду:
   ```bash
   mvn test
